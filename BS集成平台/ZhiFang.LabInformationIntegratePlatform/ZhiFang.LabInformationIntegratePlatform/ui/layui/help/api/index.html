悬浮功能  
帮助文档右下角悬浮显示目录+返回顶部功能

1、引入util.js文件，文件地址：help/util.js

2、在需要定位锚点的地方加上以下代码，效果参见页面右侧目录悬浮框

				<a class="editor-anchor" name="1" anchor-name="锚点1"></a>

name为定位的唯一标识，必须唯一，anchor-name为显示的锚点名称，util文件会自动根据锚点class="editor-anchor"样式生成目录及点击事件
