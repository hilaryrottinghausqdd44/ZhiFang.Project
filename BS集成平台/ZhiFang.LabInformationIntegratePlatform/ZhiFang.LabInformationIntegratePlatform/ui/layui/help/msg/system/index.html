危急值消息-帮助文档

消息上报

1、根据条件查询出各种类型的消息（主要用于危急值）；

2、选择其中一条消息，自动显示该条消息对应的科室和电话号码（电话号码保存在科室表中）

3、手工打电话给科室，通报该条消息的内容给科室的人员（医生或者护士）。拨打电话后选择接电话人，点击上报按钮完成上报操作；

4、上报信息保存在消息表中，包括接电话人，时间，上报人，上报标志等；

5、接电话人选项有是否只显示发送科室人员选项；

6、常用查询条件有小组、科室、就诊类型等；

上报的列表条件：已超时+未上报？

消息确认

1、在客户端消息列表中，双击或者点击确认按钮弹出确认操作界面；

2、消息确认操作界面需要显示：消息描述、消息对应原始记录结果（危机值对应样本的结果、参考值范围、样本状态等）

3、选择通知医生，输入确认人帐号、密码，点击确认

4、确认时将通知医生、确认人、确认标志、时间记录在消息表中

消息处理

1、处理界面和确认界面类似，需要显示消息描述和原始记录；

2、输入处理意见，可以从常用短语中进行快速选择；

3、处理人输入账号、密码进行处理
