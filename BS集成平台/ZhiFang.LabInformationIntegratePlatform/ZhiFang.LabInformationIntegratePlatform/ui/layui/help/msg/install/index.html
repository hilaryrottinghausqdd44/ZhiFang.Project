危急值消息-实施文档

该文档详细说明危急值消息功能实施的场景，实施的注意事项，第三方使用的说明。
服务器环境准备  
应用服务器环境准备

1、服务器操作系统要求windows server 2008及以上版本；

2、服务器需要.net framework4.7.1以上，如果没有安装请手动安装

windows server 2008 sp1请安装.net framework4.7.1；

windows server 2012 sp1请安装.net framework4.7.2；

3、安装文件请解压net472.rar，具体如何安装请百度；

数据库服务器环境准备
1、数据库版本要求sqlserver2008及以上版本；
软件概要介绍  
功能概要说明

危急值功能包含"危急值上报"、"危急值查询"、"危急值超时处理"、"科室医生确认处理"四个功能;

危急值上报：检验科产生危急值，手工打电话给科室，通报该条消息的内容给科室的人员（医生或者护士）；

危急值查询：根据过滤条件进行查询；

危急值超时处理：检验科对超时未确认的消息进行处理，相当于消息再次发送，但需要电话通知并进行记录；

科室医生确认处理：科室医生根据推送过来的危急值进行相应确认处理；

部署配置  
部署及配置说明
  • 
    开始
  • 

    部署集成平台项目

    存放ZhiFang.LabInformationIntegratePlatform项目文件夹；
    根据ZhiFang.LIIP.bak数据备份文件创建数据库；
    在IIS中部署ZhiFang.LabInformationIntegratePlatform项目；
    更改webconfig文件的数据库配置内容，将地址指向创建好的数据库ZhiFang.LIIP；

  • 

    部署危急值项目

    存放ZhiFang.LabStar.TechnicianStation项目文件夹；
    在IIS中部署ZhiFang.LabStar.TechnicianStation项；
    更改webconfig文件的数据库配置内容，将地址指向已存在的LIS数据库；

  • 
    结束
