计算公式格式说明

1.参与计算的项目用中括号将该项目的项目索引括起表示
2.加减乘除分别用+ - * / 表示
3.年龄参与计算时，用[年龄]计入公式；身高参与计算时，用[身高]计入公式；
体重参与计算时，用[体重]计入公式
4.ABS：返回给定数字表达式的绝对值 例如，ABS(-1.0)＝1, ABS(1.0) ＝1
5.EXP：返回所给的 float 表达式的指数值
例如，EXP (3.1) = 22.198, EXP (1) = 2.718
6.Ln：返回给定 float 表达式的自然对数
例如，ln(2.718) = 1, ln(11) = 2.4
7.Log10：返回给定 float 表达式的以 10 为底的对数。
例如，Log10(100) = 2 Log10(145.17) = 2.16
8.LogN：返回给定 float 表达式的以 N为底的对数。
例如，LogN(2,8) = 3 LogN(2,16) = 4
9.Sqrt：返回给定表达式的平方根,例如，Sqrt(4) = 2, Sqrt(9) = 3
10.Sqr：返回给定表达式的平方,例如，Sqr(4) = 16, Sqr(3) = 9
11.Power：返回给定表达式乘指定次方的值
例如，Power(2,3) = 8, Power(4,3) = 64, Power(4,0.5) = 2, Power(64,(1/3)) = 4, Power(4,-1)=0.25
12.Min：返回两个比较值中最小的值, 例如，Min(1,2)=1
13.Max：返回两个比较值中最大的值 ,例如，Max(1,2)=1
14.如果以上函数中的某个计算项为公式，请用()将该公式括起来
例如，141*(Power((Min(([100010]/0.7),1)),-0.329))*(Power((Max(([100010]/0.7),1)),-
1.209))*(Power(0.993,[年龄]))*1.018
