部署配置  
部署及配置说明
  • 
    开始
  • 

    部署集成平台项目

    存放ZhiFang.LabInformationIntegratePlatform项目文件夹；
    根据ZhiFang.LIIP.bak数据备份文件创建数据库；
    在IIS中部署ZhiFang.LabInformationIntegratePlatform项目；
    更改webconfig文件的数据库配置内容，将地址指向创建好的数据库ZhiFang.LIIP；

  • 

    部署危急值项目

    存放ZhiFang.LabStar.TechnicianStation项目文件夹；
    在IIS中部署ZhiFang.LabStar.TechnicianStation项；
    更改webconfig文件的数据库配置内容，将地址指向已存在的LIS数据库；

  • 
    结束
部署注意事项
1、表SC_MsgType存在默认数据[1,危急值]； 2、表IntergrateSystemSet存在默认数据[1001,智方检验之星]； 3、LIS库的科室表需要维护HIS科室编码，不然上报的危急值信息无法推到HIS登录的用户；
HIS调用说明  
http://localhost/ZhiFang.LabStar.TechnicianStation/ui/layui/interface/cv/msg.html?Account=hujie&PWD=123456&Name=胡杰&Sex=男&DeptName=前处理组&DeptHISCode=1&Phone=13812345678
其中 http://localhost/ZhiFang.LabStar.TechnicianStation为危急值项目地址，需要更改为实际部署地址；
ui/layui/interface/cv/msg.html为HIS的HTML接口地址，不需要改变；
Account=hujie&PWD=123456&Name=胡杰&Sex=男&DeptName=前处理组&DeptHISCode=1&Phone=13812345678为参数；
其中 Account=HIS账号 PWD=HIS密码 Name=用户名字 DeptName=HIS科室 DeptHISCode=HIS科室编码 Phone=用户手机号
