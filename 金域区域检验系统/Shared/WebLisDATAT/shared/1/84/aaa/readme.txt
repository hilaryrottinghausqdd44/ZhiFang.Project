     跨平台信息交流，基于 TCP/IP(UDP)，无需服务器，简单易用
     可传送文件和文件夹
     通讯数据采用 RSA/Blofish 加密

  1. 系统要求

   - MS Windows 95/98/Me/NT3.51/NT4.0/2000/XP
   - 需要 TCP(UDP)/IP 协议支持
   - comctl32.dll 要求4.71或以上版本
   - 注意: Windows 3.x 和 NT3.50, 只能使用 16 位版的 IP Messenger
     苹果机用户,则只能使用苹果版的 IP Messenger

  2. 使用说明

  □ 程序装载

   - 安装或卸载：执行 setup.exe

  □ 任务栏图标

   - 双击程序任务栏图标，即可打开发送窗口

   - 在任务栏图标上点右键可进入 [ 服务设置 ]、[ 离开 ]

  □ 发送窗口

   - 发送信息时，若勾选了 [ 封装 ] , 则收信人要先打开信封才能看到信息，
     如果你还勾选了 [ 上锁 ] ，则收信人在打开信封时还需要输入密码

   - 若要发送文件或文件夹，可直接将文件或文件夹拖入发送窗口
     或在发送窗口上点右键，再选择发送文件或发送文件夹

   - 若用户名之前有“:” ，则表示用户处于离开模式

   - 若用户名之前有“|” ，则表示用户使用的不是2.0以上版本的 IP Messenger
     将不支持文件（夹）传送和通信数据加密

   - 在发送窗口上点右键, 可以进入字体设置、显示优先值设置、
     列表显示设置，以及传送文件或文件夹

  □ 接收窗口

   - 在接收窗口上点右键, 可以进入字体设置

   - 接收窗口标题栏中的[ + ]或[ - ]表示通信数据使用了加密算法
     [ + ] 表示 RSA/1024 位、blowfish/128 位加密
     [ - ] 表示 RSA/512 位、RC2/40 位加密

   - 其它功能都很容易理解，你试一试就明白了

 ------------------------------------------------------------------------
  3. 广域网设置（广播设置）

   - 主机号全部为 1 的 IP 地址，即广播地址
     例如，连接到一个 C 类子网（即 24 位网络号，8 位主机号），IP 地址为
     aaa.bbb.ccc.ddd
     其广播地址即为: aaa.bbb.ccc.255
     若对方处于另一个私有子网中，广播可能无效

   - 更多问题，请参阅相关资料或咨询你的网络管理员

   - 若两台主机的连接经过了多个路由器，请直接指定对方 IP

   - 拨号上网用户请勾选 [ 拨号连接 ]
     当刷新在线用户列表时，列表不会被清空

  4. 其它信息

   - 所有的设置信息都保存在注册表的以下位置：
     \\HKEY_CURRENT_USER\Software\HSTools\

   *******************************************************************
   * 如果你忘记了密码, 可以在注册表中将此键值删除                    *
   * \\HKEY_CURRENT_USER\Software\HSTools\IPMsgEng\PasswordStr       * 
   *******************************************************************

   - 本软件使用的默认端口为 2425
   - 只有在启动或退出程序、使用离开模式、刷新在线用户时飞鸽传书才会进行
     消息广播
   - 本软件由 Microsoft Visual C++ 6.0 编译
   - 本软件为自由软件，你可以随意传播  :-)

 ------------------------------------------------------------------------
  5. 补充说明

   - 启动飞鸽传书前，你可以指定其运行时使用的端口，且可以使用不同的端口
     打开多个窗口。
     用法如下：
     ipmsg.exe 2426
     你可在快捷方式上设置
     但是你只能与同时也使用该端口的用户通信

   - 所以你尽可选用你喜欢的端口运行本软件，我们推荐使用 1024 端口
     介于 10000 至 20000 可能更安全些
     知名的网络文件系统软件一般使用 2049 端口
     你也可以咨询你的网络管理员

   - 如果有多个网卡(IP)，你可以将飞鸽传书与指定的网卡(IP)进行绑定
     命令格式如下（你可以在快捷方式上设置）：
     ipmsg.exe [端口] /NIC IP地址
     例如：
     C:\>ipmsg.exe /NIC 192.168.10.100

   - 支持命令方式发送信息
     命令格式如下：
     ipmsg.exe [端口] /MSG [/LOG][/SEAL] <主机名或IP地址> <消息>
     例如：
     C:\>ipmsg.exe /MSG /SEAL localhost Hello.

   - 操作技巧：

     1. 隐藏/显示 窗口  Ctrl + D

     2. 按住 Ctrl 键再点“刷新” 可保持现有用户，搜索新上线的用户

     3. 打开发送/接收窗口  Ctrl + Alt + S / R （需要进行详细设置）

     4. 打开搜索窗口  Ctrl + F

     5. 接收到多个文件，保存时勾选 [ 全部 ] 可以更方便
